
# Ghostboards

Modern Quantum Minimalism temalı Android klavye uygulaması.

## Özellikler

- 🎨 **Modern Quantum Minimalism Teması**: Saf siyah arka plan, premium gradyanlar ve mor/mavi tonları
- ⌨️ **Temel Klavye İşlevleri**: Harfler, sayılar, silme, enter, boşluk tuşları
- 🔄 **Shift Desteği**: Büyük harf yazma
- 🇹🇷 **Türkçe ve İngilizce Desteği**: Çoklu dil alt türleri
- ⚙️ **Ayarlar Ekranı**: Klavyeyi etkinleştirme ve varsayılan yapma

## Kurulum

1. Projeyi Android Studio ile açın
2. Gradle senkronizasyonunu tamamlayın
3. Uygulamayı bir Android cihazına veya emülatöre yükleyin
4. Ayarlar uygulamasından "Quantum Keyboard"ü etkinleştirin ve varsayılan yapın

## GitHub &amp; APK Oluşturma

1. Depoyu GitHub'a yükleyin
2. GitHub Actions veya lokal olarak Release APK oluşturun
3. `./gradlew assembleRelease` komutu ile release build'i alın

## Teknik Bilgiler

- **Dil**: Kotlin
- **Minimum SDK**: 24 (Android 7.0)
- **Hedef SDK**: 34 (Android 14)
- **Gradle**: 8.2
